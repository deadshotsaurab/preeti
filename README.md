# 🎨 Preetilata Fine Art Portfolio

A responsive and elegant portfolio website showcasing the artworks, achievements, and artistic journey of **Preetilata**, an MFA Gold Medalist from BHU.

## 🌟 Features

- Beautiful landing page with hero section
- Artwork gallery showcase
- Artist profile section
- Admin panel for artwork management
- Responsive design for mobile, tablet, and desktop
- Smooth animations and interactive UI

## 🛠 Tech Stack

- HTML5
- CSS3
- JavaScript
- JSON (for artwork data)

## 📂 Project Structure

```bash
preeti/
│
├── images/              # All artwork and asset images
├── index.html           # Main portfolio page
├── admin.html           # Admin panel
├── style.css            # Styling
├── script.js            # JavaScript functionality
├── artworks.json        # Artwork data
└── README.md
```


## 💻 Installation

Clone repository:

```bash
git clone https://github.com/deadshotsaurab/preeti.git
```

Open project folder:

```bash
cd preeti
```

Run locally by opening:

```bash
index.html
```
