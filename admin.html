🚀 How to Add New Work & Go Live

1

Copy your painting image into the folder

Put your new painting photo/scan inside the images/ folder (same folder as index.html). Name it something simple like my_new_work_2025.jpg

2

Use the "Add New Work" tab

Fill in the title, year, medium, size, description and the image filename. Click Add to Gallery. The new work appears on your portfolio instantly!

3

Download the updated artworks.json

Click the button below to download your updated artwork list. Replace the old artworks.json file in your portfolio folder with this new one.


4

Upload to your hosting (make it live)

Upload the entire portfolio folder to any free hosting:

• Netlify (netlify.com) — drag & drop the folder, free forever

• GitHub Pages — free with a GitHub account

• Vercel (vercel.com) — free, very fast

5

Keep this admin page for future updates

Whenever you finish a new painting — drop the image in the folder, use this admin panel to add it, download the new artworks.json, and re-upload. Done! 🎨

💡 Quick Tip: This admin page (admin.html) stays on your computer. You don't need to upload it to the live site. Only upload index.html, style.css, script.js, artworks.json, and the images/ folder.
